Willkommen auf meiner privaten Homepage

Privater Bereich
Diese Seite ist nicht öffentlich zugänglich. Um den Inhalt der Seiten in diesem und allen darunterliegenden Verzeichnissen zu sehen, müssen Sie Ihren Benutzernamen und Ihr Passwort eingeben.

Der Zugang wird durch die Datei ".htaccess.n" in homepage/private gesteuert.

Die Datei hat den folgenden Aufbau:

AuthType Basic
AuthName "My private stuff"
require user UserListe
Die UserListe enthält die nethz-Benutzernamen der Personen, welche auf dieser Seite einloggen können. Die einzelnen Benutzernamen müssen durch ein Leerzeichen getrennt sein.

Anleitung zur Zugangsbeschränkung Ihrer Homepage finden Sie auf http://www.id.ethz.ch/services/list/stud_homepage/.

Wichtige Hinweise:
AuthType Basic, AuthName und require user sind reservierte Worte und müssen unverändert bleiben!
Windows Benützer müssen einen Editor verwenden, der das Unix-Zeilenumbruch-Format abspeichern kann (Bsp. SciTE).
Sorgfalt und Tests erhöhen die Sicherheit vor unberechtigten Zugriffen. Beenden Sie Ihren Web-Browser, wenn Sie den Zugriff auf passwortgeschützte Bereiche nicht mehr benötigen.

Im Falle von unüberwindlichen Problemen erhalten Sie hier Support.
